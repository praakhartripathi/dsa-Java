## ðŸ§© Problem Description

Given a binary string `s`, return the number of non-empty substrings that have:

* Equal number of `0`s and `1`s
* All `0`s grouped consecutively
* All `1`s grouped consecutively

Substrings that occur multiple times are counted multiple times. ([leetcode.com][1])

## ðŸš€ Approach (Greedy + Group Counting)

### ðŸ”‘ Key Observation

Valid substrings are formed only between **adjacent groups** of consecutive characters.

Example:

```
00110011
```

Groups:

```
00 | 11 | 00 | 11
```

Lengths:

```
[2, 2, 2, 2]
```

Each adjacent pair contributes:

```
min(previousGroup, currentGroup)
```

---

### ðŸ§  Algorithm Steps

1. Traverse the string once.
2. Maintain:

   * `prev` â†’ length of previous group
   * `cur` â†’ length of current group
3. If characters are same â†’ increase `cur`.
4. If character changes:

   * Add `min(prev, cur)` to answer.
   * Update `prev = cur`.
   * Reset `cur = 1`.
5. Add last `min(prev, cur)` after loop.

## ðŸ’» Java Solution

```java
class Solution {
    public int countBinarySubstrings(String s) {

        int prev = 0;
        int cur = 1;
        int ans = 0;

        for (int i = 1; i < s.length(); i++) {

            if (s.charAt(i) == s.charAt(i - 1)) {
                cur++;
            } else {
                ans += Math.min(prev, cur);
                prev = cur;
                cur = 1;
            }
        }

        ans += Math.min(prev, cur);
        return ans;
    }
}
```

## ðŸ§  Intuition Summary

* Break the string into consecutive groups.
* Only adjacent groups can form valid substrings.
* The number of valid substrings equals the minimum size of neighboring groups.
